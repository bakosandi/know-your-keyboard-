## Brackets

Name | Symbol | Shortcut
-- | --- | ---
Round brackets aka parentheses | ( ) | shift+8;9
Square brackets aka box brackets | [ ] | alt gr+f;g
curly brackets aka curly braces | { } | alt gr+b;n
angle brackets aka chevrons | < > | alt gr+í;y
